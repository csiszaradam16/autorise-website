## Claude Code Implementation Notes

```
PROJECT: Autorise Marketing Website
STACK SUGGESTION: Next.js or Astro + Tailwind CSS
DESIGN REFERENCE: surferseo.com (clean, bold, dark sections, strong typography)

KEY REQUIREMENTS:
1. Mobile-first responsive design
2. Dark hero section with gradient or subtle animation
3. Alternating light/dark sections for visual rhythm
4. Strong typography hierarchy (large headlines, readable body)
5. Smooth scroll animations (subtle, not distracting)
6. Fast loading (optimize images, minimal JS)
7. Contact form with email integration
8. SEO-optimized (meta tags, schema markup, fast Core Web Vitals)

SECTIONS IN ORDER:
1. Navigation (sticky, minimal)
2. Hero (dark background, headline, CTAs, trust bar)
3. Problem statement (light background)
4. Services overview (two main blocks, dark background)
5. Why Autorise (4 cards, light background)
6. How it works (3 steps, could be dark)
7. Testimonials (light background, carousel optional)
8. Stats bar (could be integrated or standalone)
9. Final CTA (dark, bold)
10. Footer

INTERACTIONS:
- Smooth scroll to sections from nav
- Hover states on cards and buttons
- Form validation on contact form
- Optional: Subtle parallax on hero
```
